Beschrijving

Status en fout codes:

40xx: Entitlement API

41xx: Delivery API

60xx: Order API

Toevoegen van foutcodes die een ontvanger kan teruggeven om aan te geven dat een ontvanger een bepaalde levervariant niet wordt ondersteund.

Dit is van toepassing op referentiecomponenten aanspraakmanagers, licentieregistraties en portalen en de Entitlement en DeliveryOrder variant.

Toe te voegen foutmeldingen

Toe te voegen foutmeldingen (op de xx moet een cijfer  komen):

40xx : Levervariant wordt niet ondersteund

41xx: Aanspraakvariant wordt niet ondersteund.

60xx: Ordervariant wordt niet ondersteund

Onderbouwing van nut of noodzaak

Indien een systeem een duidelijke foutmelding terugkrijgt op het verzenden van een bericht kan deze het betreffende proces beter monitoren.
